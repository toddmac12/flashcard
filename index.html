Javascript (JS)

JavaScript is a programming language used to make web pages interactive.

Operators

+, -, *, /, and % are called operators. An operator is a special character (or characters) that indicates an action to be performed.

Variables

A variable is just a container that stores some information such as a number, an object, or a string.

Variable naming conventions

Use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word. Variables should begin with a letter, are case sensitive and should have clear names.

Functions

Functions are one of the fundamental building blocks in JavaScript. A function in JavaScript is similar to a procedureâ€”a set of statements that performs a task or calculates a value, but for a procedure to qualify as a function, it should take some input and return an output where there is some obvious relationship between the input and the output. To use a function, you must define it somewhere in the scope from which you wish to call it.

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

Some methods take one or more arguments that provide the method with additional information to help it perform its action.

Parameters

A parameter is a variable that's assigned to an argument.

Return

The return value is the method's response.

Chaining Methods

Calling a method directly on the return value of another method.

Strings

A string is a set of characters enclosed in quotations. "hello " is a string. So is "Good morning, everyone!!! " And, just to make things a little tricky, "21 " is a string, not a number. That's because it's enclosed in quotations, and any set of characters - whether that's letters, numbers, nineteen exclamation points in a row, or some combination thereof - is a string.

Booleans

A boolean can be one of two values: true or false. That's it.

Undefined

This means something hasn't been defined in our code yet.

NaN

Not a number.

Null

Null just means nothing. We will sometimes want to assign a null value to a thing because we don't know what value it should have yet.

Attributes

These include classes and IDs, which describe what code may apply to an element.

Comments

anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

Alert

The alert() method displays an alert box with a specified message and an OK button.

jQuery

jQuery is a JavaScript library that makes it easier to incorporate JavaScript on your web pages.
